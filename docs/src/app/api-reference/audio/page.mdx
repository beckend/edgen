# Audio

Discover how to convert audio to text or text to audio. OpenAI compliant. {{ className: 'lead' }}

---

## Create transcription {{ tag: 'POST', label: 'http://localhost:33322/v1/audio/transcriptions' }}

<Row>
  <Col>

    Transcribes audio into text.

    ### Required attributes

    <Properties>
      <Property name="username" type="string">
        The username for the contact.
      </Property>
      <Property name="phone_number" type="string">
        The phone number for the contact.
      </Property>
    </Properties>

    ### Optional attributes

    <Properties>
      <Property name="avatar_url" type="string">
        The avatar image URL for the contact.
      </Property>
      <Property name="display_name" type="string">
        The contact display name in the contact list. By default, this is just the username.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
